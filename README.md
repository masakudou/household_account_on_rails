# 無駄遣いチェッカー

本アプリは、プログラミング学習の一環として制作している、Ruby on railsフレームワークを用いた家計簿アプリです。

## 本アプリの機能一覧
### 実装済み

- ユーザー登録機能(メールアドレス)

- ホーム画面へのアイコン画像とヘッダー画像の登録機能

- カレンダーを用いた収入・支出の記録(コメント付き)

- 統計機能

## 使用した技術
- Container: Docker for Mac
  - Web Application Server
    - Ruby 2.6.3
    - Ruby on Rails 5.2.3
  - DB Server
    - PostgreSql
  - DB GUI
    - pgweb
  - CI/CD
    - CircleCI

- Test Framework: Rspec
- Cloud Service: heroku
